# Unbound Force

An AI agent swarm themed as a superhero team. Engineers
shift from manual coding to directing AI agents through
specifications and governance — treating specs and rules
as the medium through which human intent is manifested
into code.

## Getting Started

```bash
brew install unbound-force/tap/unbound-force
uf setup
uf init
```

See the [unbound-force](https://github.com/unbound-force/unbound-force)
repo for full documentation.
